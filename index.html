Night Dream : Audio Creation Stack

Documentation et ressources pour la création musicale indépendante à la guitare.

Description du Projet

Ce dépôt GitHub centralise les recherches et les méthodes d'artisanat sonore développées par Soochrys. Le projet Night Dream sert de base d'étude pour démontrer qu'une production instrumentale de haute qualité est accessible avec une approche pragmatique et épurée.
